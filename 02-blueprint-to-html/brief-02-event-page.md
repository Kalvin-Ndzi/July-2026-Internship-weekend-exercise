# Brief 02 — Event Page

Build a complete HTML page announcing an event (real or invented — a school hackathon, a local meetup, a concert, anything). **Blank file. No starter HTML.**

## The page needs

- A clear page title in the browser tab
- A header area with the event name and date
- A navigation area linking to at least 3 sections further down the same page (using anchor links)
- A "Details" section: what, when, where — presented however you think best communicates it
- A "Schedule" section, presented as a table (time in one column, activity in another)
- An "RSVP" section containing a real form: name, email, and a dropdown for "which day are you attending" (if it's multi-day) or a radio group for "will you attend" (yes/no/maybe)
- A "Location" section with an embedded map (`<iframe>` from a real map provider)
- A footer with contact info

## Constraints

- No starter HTML provided.
- Every form input needs a properly linked `<label>` — check the `for`/`id` match yourself before submitting.
- The `<iframe>` needs a `title` attribute.
- Anchor links in your navigation must actually jump to the right sections when clicked — test them.

## Self-check before you submit

- [ ] Click every nav link — do they jump to the correct section?
- [ ] Click every `<label>` in the form — does it focus the right input?
- [ ] Does the schedule use a real `<table>`, not a list pretending to be one?
- [ ] Does your iframe have a `title`?
