# Brief 03 — Personal Portfolio Page

Build a one-page personal portfolio/about page from scratch. **Blank file. No starter HTML.**

## The page needs

- A page title in the browser tab with your name
- A header with your name and a short tagline
- A navigation area linking to at least 3 sections below (About, Projects, Contact)
- An "About" section — a couple of paragraphs about yourself
- A "Projects" section listing at least 3 projects (real or invented), each with a name, short description, and a link (use `<article>` for each project — they're self-contained)
- A "Skills" section — pick the list type that actually fits (does order matter here or not?)
- A "Contact" section with a real contact form: name, email, message, and a submit button
- A footer with a copyright line and a mailto link

## Constraints

- No starter HTML provided.
- No `<div>` tags — use semantic tags for every region.
- Every heading must be in correct order, no skipped levels.
- Every form input needs a matched `<label for>`/`id` pair.

## Self-check before you submit

- [ ] Does the page have exactly one `<h1>`?
- [ ] Do your 3 project entries use `<article>`, not `<div>`?
- [ ] Does clicking each nav link jump to the right section?
- [ ] Does the form actually work when you tab through it with only your keyboard?
