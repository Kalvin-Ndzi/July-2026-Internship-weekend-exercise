# Brief 01 — Recipe Page

Build a complete HTML page for a recipe from scratch. **Do not start from a template — start from a blank file.** Use your own recipe (something you actually know how to make) rather than lorem ipsum, if possible.

## The page needs

- A page title (browser tab) that names the actual dish
- A main heading with the recipe name
- A short intro paragraph describing the dish
- An image of the finished dish, with meaningful `alt` text
- A list of ingredients — think about which list type actually fits (does order matter?)
- A list of preparation steps — think about which list type fits *this* one
- Prep time and cook time, presented as a small table
- A "Notes" or "Tips" section
- A footer with your name as the author

## Constraints (this is the actual test)

- No starter HTML is provided. You choose every tag.
- You may not use a single `<div>` anywhere on this page. If you're tempted to reach for one, there's almost always a semantic tag that fits better — figure out which.
- Every image must have real, descriptive `alt` text.
- Headings must be in correct, unskipped order.

## Self-check before you submit

- [ ] Are your two lists using different tags if their content actually needs it?
- [ ] Did you use `<table>` for the time data, not for layout?
- [ ] Zero `<div>` tags in the file?
